== Description ==

Woocommerce Product addon plugin. Add custom fields to your Woocommerce product page. With an easy-to-use Custom Form Builder, now you can add extra product options quickly.
Custom Product Options or fields can be added to product ordering page. This extra form data submitted by customers will be shown in site backend along with order details.

= Features =
This extra product option plugin supports following form elements
* <strong>Text</strong> input field
* <strong>Number</strong> input field
* <strong>Radio</strong>  field
* <strong>Checkbox</strong>  field
* <strong>Select</strong> input field
* <strong>Textarea</strong>  field
* <strong>Hidden</strong> input field
* <strong>Upload Field</strong>

And apart from above form elements Woocommerce Custom Product Addons supports following html tags
* <strong>Paragraph (p)</strong> tag
* <strong>Header (h)</strong> tag, H1-H6

== Installation ==

Installing "Woocommerce Product Addons" can be done either by searching for "Woocommerce Product Addons ( Custom fields to add extra product options )" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
